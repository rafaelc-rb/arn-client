Política de Privacidade - Aranet Provedor

Última atualização: 04 de novembro de 2025

1. Sobre o Aplicativo

O aplicativo Aranet Provedor permite que clientes acessem e visualizem suas faturas de serviço de internet através de dispositivos móveis.

2. Coleta de Dados

Este aplicativo funciona como um portal de acesso web (WebView) e não coleta, armazena ou processa dados pessoais diretamente no dispositivo. Todos os dados são gerenciados pela plataforma web Aranet em seus servidores.

3. Permissões Solicitadas

  • INTERNET: Necessária para acessar a plataforma web através do navegador integrado.

4. Uso de Dados

O aplicativo apenas exibe conteúdo da plataforma web Aranet GO. Dados de login, informações de faturas e dados pessoais são processados pela plataforma web conforme sua própria política de privacidade.

5. Segurança

O aplicativo utiliza conexões HTTPS seguras, garantindo a criptografia dos dados em trânsito entre o dispositivo e os servidores da Aranet.

6. Cookies e Armazenamento

O aplicativo pode armazenar cookies de sessão necessários para o funcionamento da plataforma web. Estes são gerenciados automaticamente pelo componente WebView do Android.

7. Compartilhamento de Dados

Este aplicativo não compartilha dados com terceiros. Todos os dados permanecem entre o usuário e a plataforma Aranet.

8. Direitos do Usuário (LGPD)

Conforme a Lei Geral de Proteção de Dados (LGPD), você tem direito a:

  • Acessar seus dados pessoais
  • Corrigir dados incompletos ou desatualizados
  • Solicitar a exclusão de dados
  • Revogar consentimento

Para exercer esses direitos, entre em contato através da plataforma web ou pelos canais oficiais da Aranet Provedor.

9. Alterações nesta Política

Reservamos o direito de atualizar esta política de privacidade. Alterações serão publicadas nesta página com a data de atualização revisada.

10. Contato

Para dúvidas sobre esta política de privacidade ou sobre o tratamento de dados:

  • Site: https://aranetgo.sgplocal.com.br

11. Legislação Aplicável

Esta política está em conformidade com a Lei Geral de Proteção de Dados (LGPD - Lei 13.709/2018) do Brasil.
